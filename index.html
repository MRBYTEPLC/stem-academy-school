🌿 IF / ELSE SETUP

What is IF/ELSE? Checks ONE condition → if TRUE runs the IF block, if FALSE runs ELSE. Only one branch runs!
IF (today == Saturday OR Sunday):
→ Stay home 🏠
ELSE:
→ Go to work 💼

🔄 FOR LOOP SETUP

What is a FOR Loop? Repeats an action an exact number of times. Use it when you know in advance how many repetitions you need.
Example: Start=1, End=5, Step=1
FOR i=1: do action
FOR i=2: do action
FOR i=3: do action
FOR i=4: do action
FOR i=5: do action ← STOPS here
Total: 5 iterations

🔄 WHILE LOOP SETUP

What is a WHILE Loop? Checks condition FIRST. If FALSE at start → loop body NEVER runs! Loops only while condition stays TRUE.
Example: Start=0, Limit=4, Step=1
WHILE (0 < 4): run → counter=1
WHILE (1 < 4): run → counter=2
WHILE (2 < 4): run → counter=3
WHILE (3 < 4): run → counter=4
WHILE (4 < 4)? NO → EXIT loop
Total: 4 iterations

Press EXECUTE to see the trace...

🧱 Now Try It With Blocks!

💡 Label your switch "still_sleeping". TRUE → loops forever, FALSE → 0 runs.

🔄 DO-WHILE SETUP

What is DO-WHILE? Runs the body FIRST, then checks. Always executes at least 1 time even if condition starts FALSE!
Example: Start=1, Limit=3
DO: action (1st run, no check yet!)
WHILE (1 ≤ 3)? YES → continue
DO: action (2nd run)
WHILE (2 ≤ 3)? YES → continue
DO: action (3rd run)
WHILE (3 ≤ 3)? YES → continue
DO: action (4th run)
WHILE (4 ≤ 3)? NO → EXIT
Even if Start=99: runs ONCE first!

🏦 THE ATM MACHINE — Nested IF

What is Nested IF? An IF statement placed INSIDE another IF. The inner IF only runs if the outer condition is TRUE first.
IF pin == "1234":
IF cash <= balance: DispenseCash
ELSE: ShowError("Insufficient funds")
ELSE: ShowError("Incorrect PIN")

🚦 SMART TRAFFIC LIGHT — IF/ELSEIF/ELSE

What is IF/ELSEIF/ELSE? Checks conditions IN ORDER. The first TRUE one runs — all the rest (including ELSE) are skipped.
IF light == "Red": Brake
ELSEIF light == "Yellow": SlowDown
ELSE: DriveForward

❄️ SMART AIR CONDITIONER — AND Logic

What is AND Logic? BOTH conditions must be TRUE at the same time for the result to be TRUE.
IF temperature > 25 AND empty_room == FALSE:
TurnOnAC
ELSE: TurnOffAC

🤖 COLOR-SORTING ROBOT — TEST/CASE

What is TEST/CASE? Compares ONE value against several CASEs. The first match wins; if nothing matches, DEFAULT runs.
TEST detected_color
CASE "Red": MoveToRedBin
CASE "Blue": MoveToBlueBin
DEFAULT: MoveToScrapBin
